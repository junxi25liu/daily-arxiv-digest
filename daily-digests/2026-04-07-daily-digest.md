1. **六维度情感标注体系**: 包含情感极性、开放词汇情感描述、强度等级、韵律属性、显著片段和语义内容，实现多粒度语音情感建模
2. **人 -LLM 协作标注流程**: 结合算法预标注、多 LLM 描述生成和人工验证，平衡标注质量和可扩展性
3. **多样化描述风格**: 将标注重构为多种描述风格，增强语言多样性并减少下游建模中的风格偏差

### 🔧 技术方法详解

**标注维度**:
- **情感极性**: 正面/负面/中性

- **韵律属性**: 音高、语速、响度等韵律特征描述
- **显著片段**: 情感表达最突出的时间段
- **语义内容**: 语音文本内容

**标注流程**:
```
原始语音 → 算法预标注 → 多 LLM 描述生成 → 人工验证 → 风格重构 → 最终标注
```

**人 -LLM 协作**:
- 算法预标注提供初始情感标签
- 多个 LLM 生成多样化描述
- 人工验证确保质量
- 风格重构增加语言多样性

### 📊 实验结果摘要

**语音情感描述生成**:

- 开放词汇描述生成质量显著提升

**情感语音合成**:

## 论文 2: DynFOA: Generating First-Order Ambisonics with Conditional Diffusion for Dynamic and Acoustically Complex 360-Degree Videos

**arXiv**: [2604.02781](https://arxiv.org/abs/2604.02781)  
**作者**: Ziyu Luo, Lin Chen, Qiang Qu, Xiaoming Chen, Yiran Shen  
**机构**: Academic  
**分类**: cs.SD  
**提交日期**: 2026-04-03

### 📌 一句话总结

提出 DynFOA 框架，结合动态场景重建与条件扩散模型，为动态和声学复杂的 360 度视频自动生成一阶 Ambisonics (FOA) 空间音频，解决 360 度视频空间音频捕获困难的问题。

### 💡 核心创新点

1. **360 度视频空间音频生成**: 使用扩散模型为 360 度视频生成 FOA 空间音频
2. **动态场景处理**: 能够处理视频中声源位置和环境的动态变化
3. **3D Gaussian Splatting 场景重建**: 使用 3DGS 重建场景几何和材质，提供物理基础的特征
4. **多条件扩散生成**: 将视觉特征、深度、语义、声源位置等多模态信息作为扩散模型条件

### 🔧 技术方法详解

**整体流程**:
```
360 度视频 → 动态声源检测与定位 → 深度与语义估计 → 3DGS 场景重建
                                                      ↓
                                              物理声学特征提取
                                                      ↓
                                              条件扩散模型
                                                      ↓
                                              FOA 空间音频 (W,X,Y,Z)
```

**关键技术**:

1. **动态声源检测与定位**:
   - 分析视频帧检测运动物体
   - 估计声源位置随时间变化

2. **3D Gaussian Splatting (3DGS) 重建**:
   - 从 360 度视频重建场景几何
   - 估计材质属性 (影响声音反射和吸收)
   - 提供物理基础的声学特征

3. **条件扩散模型**:
   - 输入：噪声 + 多模态条件 (视觉、深度、声源位置、场景几何)
   - 输出：FOA 四通道音频 (W-全向，X/Y/Z-方向分量)
   - 训练：逐步去噪生成与场景一致的音频

4. **声学交互建模**:
   - 显式建模遮挡、反射、混响效果
   - 考虑声源 - 环境 - 听者视角的交互

**M2G-360 数据集**:
- 600 个真实世界片段
- 三个子集：MoveSources (移动声源)、Multi-Source (多声源)、Geometry (复杂几何)

### 📊 实验结果摘要

**客观指标**:
- **空间准确性**: 显著优于现有方法
- **声学保真度**: 在频谱和相位指标上表现更好
- **分布匹配**: 生成音频与真实音频分布更接近

**主观评估**:
- **沉浸感体验**: 用户评价显著优于基线方法
- **音画同步**: 生成音频与视频内容同步良好

**消融实验**:
- 无 3DGS 重建：性能下降明显
- 无条件扩散：空间准确性显著降低
- 完整模型：最佳性能

### 🔗 相关链接

- [arXiv 页面](https://arxiv.org/abs/2604.02781)
- [PDF 下载](https://arxiv.org/pdf/2604.02781.pdf)
- [项目页面](https://dynfoa.github.io/) (预期)

### 🤔 个人评价

DynFOA 是 360 度视频空间音频生成的有益尝试。随着 VR/AR 设备普及，360 度视频内容需求增长，但空间音频录制成本高。自动生成方法可以降低内容创作门槛。

**局限性**:
- FOA 分辨率有限，高阶 Ambisonics 效果更好但更难生成
- 复杂声学环境 (如多个移动声源) 仍有挑战
- 3DGS 重建计算成本较高

**启发**:
- 扩散模型在空间音频生成中的应用前景广阔
- 物理基础的场景重建对生成质量至关重要
- 多模态条件 (视觉 + 深度 + 运动 + 几何) 对生成质量至关重要

1. **代理式多轮时序推理**: 不是单次推理，而是迭代分析全局音频结构，自主预测时间边界
2. **联合建模**: 同时建模说话人身份、性别、时间戳和转录内容
3. **说话人感知缓存**: 扩展处理超出训练上下文窗口的长音频

### 🔧 技术方法详解

**核心挑战**:
- 语音大语言模型在单人任务上表现优秀，但多人场景仍具挑战

```

                                 ↓
```

1. **迭代推理机制**:

   - 第二轮：预测说话人转换边界
   - 第三轮：对每个片段进行细粒度分析

2. **说话人感知缓存**:
   - 维护说话人嵌入缓存
   - 支持长音频的连续处理
   - 跨片段说话人一致性建模

3. **三阶段训练**:
   - 阶段 1：基础语音识别
   - 阶段 2：说话人归属
   - 阶段 3：时间戳定位和联合推理

### 📊 实验结果摘要

- **AISHELL-4**: 多人对话数据集

**主要结果**:

- 在重叠语音处理上表现突出
- 在复杂轮替场景下保持高性能

**对比基线**:

## 论文 5: Split and Conquer Partial Deepfake Speech

**arXiv**: [2604.02913](https://arxiv.org/abs/2604.02913)  
**作者**: Inbal Rimon, Oren Gal, Haim Permuter  
**机构**: Academic  
**分类**: cs.SD, cs.AI, cs.LG  
**提交日期**: 2026-04-03

### 📌 一句话总结

提出分而治之的局部深度伪造语音检测框架，将问题分解为边界检测和片段级分类两阶段，在 PartialSpoof 和 Half-Truth 基准上取得最先进性能。

### 💡 核心创新点

1. **两阶段分解**: 将局部深度伪造检测分解为边界检测和片段级分类
2. **边界检测器**: 识别时间转换点，将音频划分为声学一致的片段
3. **反射式多长度训练**: 将可变长度片段转换为多个固定输入长度
4. **多配置融合**: 各阶段使用多种配置训练，融合互补预测

### 🔧 技术方法详解

**问题定义**:
- 局部深度伪造：仅在语音的短时间片段中被篡改
- 传统 utterance 级分类器难以检测局部篡改

**整体架构**:
```
输入音频 → 边界检测器 → 片段划分 → 片段级分类器 → 融合决策
                          ↓
                     bona fide / fake
```

**关键技术**:

1. **边界检测器**:
   - 识别声学特征突变点
   - 输出时间转换边界
   - 基于 CNN 或 Transformer

2. **片段级分类器**:
   - 对每个片段独立分类
   - 输入：固定长度片段
   - 输出：bona fide 或 fake

3. **反射式多长度训练**:
   - 将可变长度片段转换为多个固定长度
   - 通过反射填充实现
   - 产生多样化特征表示

4. **多配置融合**:
   - 不同特征提取器
   - 不同增强策略
   - 融合互补预测

### 📊 实验结果摘要

**PartialSpoof 基准**:
- 在多个时间分辨率上取得最先进性能
- 在 utterance 级检测上也取得最先进性能
- 在伪造区域检测和定位上显著提升

**Half-Truth 数据集**:
- 取得最先进性能
- 证实了方法的鲁棒性和泛化能力

**消融实验**:
- 边界检测器贡献显著
- 多长度训练提升鲁棒性
- 多配置融合进一步提升性能
