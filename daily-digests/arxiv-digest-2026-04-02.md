### 1. MambaVoiceCloning: Efficient and Expressive Text-to-Speech via State-Space Modeling and Diffusion Control

**arXiv**: [2604.00292](https://arxiv.org/abs/2604.00292)  
**提交日期**: 2026-03-31  
**分类**: cs.SD, cs.LG  
**作者**: Sahil Kumar 等  
**会议**: ICLR 2026 (已接收)

#### 一句话总结
MambaVoiceCloning (MVC) 首次实现扩散 TTS 的条件路径在推理时完全使用状态空间模型（SSM），移除所有注意力和显式 RNN 层，在保持或提升音质的同时实现线性时间复杂度和 1.6 倍吞吐提升。

#### 核心创新点
1. **纯 SSM 条件路径**: 推理时完全移除注意力和 RNN 层，仅使用 Mamba SSM 进行文本、节奏和韵律编码
2. **三组件架构**: 
   - Gated Bidirectional Mamba 文本编码器
   - Temporal Bi-Mamba（由轻量级对齐教师监督，训练后丢弃）
   - Expressive Mamba + AdaLN 调制
3. **线性时间复杂度**: O(T) 条件编码，有界激活内存，支持实用有限前瞻流式
4. **超越混合架构**: 相比 StyleTTS2、VITS 和 Mamba-注意力混合模型，在 MOS、F0 RMSE、MCD、WER 上均有提升

#### 技术方法详解
- **骨干网络**: 固定 StyleTTS2 mel 扩散 + 声码器骨干
- **编码器替换**: 将原有基于注意力的持续时间和风格模块替换为纯 SSM
- **训练策略**: 在 LJSpeech/LibriTTS 上训练，VCTK、CSS10（西/德/法）和长文本 Gutenberg 段落评估
- **效率优势**: 编码器参数降至 21M，吞吐提升 1.6 倍

#### 实验结果摘要
| 指标 | MambaVoiceCloning | StyleTTS2 | VITS | Mamba-Attention 混合 |
|------|-------------------|-----------|------|---------------------|
| MOS | 显著更高 | 基准 | 基准 | 基准 |
| F0 RMSE | 更低 | - | - | - |
| MCD | 更低 | - | - | - |
| WER | 更低 | - | - | - |
| 编码器参数 | 21M | - | - | - |
| 吞吐 | 1.6x | 1.0x | - | - |

### 2. FineLAP: Taming Heterogeneous Supervision for Fine-grained Language-Audio Pretraining

**arXiv**: [2604.01155](https://arxiv.org/abs/2604.01155)  
**提交日期**: 2026-04-01  
**分类**: cs.SD  
**作者**: Xiquan Li 等

#### 一句话总结
FineLAP 提出细粒度语言 - 音频预训练范式，通过双流 sigmoid 损失和聚类采样策略，联合学习片段级和帧级对齐，在音频理解任务上实现 SOTA 性能。

#### 核心创新点
1. **异构数据利用**: 有效利用大规模片段级文本描述和有限的帧级标注数据
2. **双流 Sigmoid 损失**: 联合学习片段级和帧级监督，粗粒度和细粒度对齐相互促进
3. **解耦音频投影器**: 在自监督编码器之上使用解耦音频投影器，同时捕获全局语义和局部细节
4. **FineLAP-100k 数据集**: 通过可扩展的策划流程构建的大规模合成 SED 数据集，缓解时序标注数据稀缺

#### 技术方法详解
- **聚类采样策略**: 平衡片段级和帧级样本的采样
- **架构设计**: 自监督编码器 + 解耦投影器，分离全局和局部特征学习
- **数据构建**: FineLAP-100k 包含 100k 合成声音事件检测样本
- **多任务评估**: 检索、分类、声音事件检测、文本到音频定位

#### 实验结果摘要
- 在多个音频理解任务上实现 SOTA 性能
- 检索、分类、SED、text-to-audio grounding 均有提升
- 消融研究证实粗粒度和细粒度对齐的互补性

### 3. Sona: Real-Time Multi-Target Sound Attenuation for Noise Sensitivity

**arXiv**: [2604.00447](https://arxiv.org/abs/2604.00447)  
**提交日期**: 2026-04-01  
**分类**: cs.SD, cs.HC  
**作者**: Jeremy Zhengqi Huang 等

#### 一句话总结
Sona 是交互式移动系统，支持实时多目标声音衰减，帮助噪声敏感人群选择性减弱烦人声音，同时保持对周围环境的感知。

#### 核心创新点
1. **多目标同时衰减**: 克服 prior systems 的单目标限制，支持同时衰减多个重叠声源
2. **目标条件神经管道**: 通过现场音频示例支持用户可扩展的声音类别，无需重新训练
3. **实时设备端运行**: 低延迟，适合实时聆听场景
4. **用户研究驱动**: 68 人形成性研究 + 10 人现场研究，验证实用性和有效性

#### 技术方法详解
- **神经架构**: 目标条件化管道，支持多声源分离和衰减
- **用户扩展**: 通过少量现场音频示例即可添加新声音类别
- **延迟优化**: 设备端实时运行，满足低延迟要求
- **评估方法**: 技术基准测试 + 现场用户研究

#### 实验结果摘要
- 低延迟、多目标衰减适合实时聆听
- 显著减少烦人声音，同时保持环境感知
- 用户可扩展声音类别，无需重新训练
- 68 人形成性研究和 10 人现场研究验证有效性
